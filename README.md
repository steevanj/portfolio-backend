# Portfolio Backend API

Backend API for **stevedev.live** built with Django and Django REST Framework.

## Tech Stack

* Django
* Django REST Framework
* MySQL
* Docker
* AWS EC2

## Features

* Blog API
* Skills API
* Portfolio API
* Contact messages
* Admin dashboard with Jazzmin

## Run Locally

```bash
docker-compose up --build
```

Visit:

```
http://localhost:8000
```

Admin panel:

```
http://localhost:8000/me
```

## Deployment

Deployed on AWS EC2 using Docker.
